# Introduction

OpenF1 is an open-source API providing detailed <span style="white-space: nowrap;">Formula 1</span> telemetry, timing, and session data in JSON and CSV formats.

Historical data (from 2023 onwards) is free and accessible without authentication. Real-time data requires <a href="https://buy.stripe.com/eVqcN41BPekP0iIalBcEw02" target="_blank">a paid subscription</a>.

You can query the API directly through your browser or HTTP client. Explore the endpoints below to get started.
